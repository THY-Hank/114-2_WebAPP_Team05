# Team5
這是Team5的weekly HW專案 
打算做一個具有聊天室的程式碼管理網站，像是部分github功能+即時聊天室
3/20 加入上傳code、看code、可以評論、聊天室、可以分享檔案
## push your code by the following command
```
git add .
git commt -m "commit message"
git checkout -b branchname
git push origin branchname
```


## before start the web you need to install the package at frontend
```
cd frontend
npm install 
```
## start the web
```
npm run dev
```
## you can use this to check your  code
```
npm run lint
```
